# ParkingNow — Owner Panel

## Product

Admin dashboard for parking lot owners in Lima. Owners monitor IoT sensor state, manage reservations, configure spaces, watch camera feeds, and track revenue — all from one authenticated web surface.

## Register

product

## Brand

- **Palette**: Midnight navy (`#040c1b`) sidebar + electric teal (`#00d4aa`) accent + warm off-white (`#F5F3EF`) content area
- **Primary**: midnight navy for actions; teal for accent, states, indicators
- **Fonts**: Space Grotesk (headings), DM Sans (body/labels), JetBrains Mono (data/timestamps)
- **Tone**: Smart · Modern · Urban — operational precision, not startup hype

## Users

Parking lot owner in Lima. Checks dashboard on laptop from parking office or home. Needs at-a-glance status, quick action on reservations, and confidence the data is live.

## Anti-references

- Generic SaaS dashboard (Notion-style cream + purple)
- Government/municipal interfaces (grey, bureaucratic)
- Startup hype (gradients everywhere, rounded corners on everything)
- Consumer apps (oversized UI, decorative animation, playful tone)

## Key surfaces

1. Sign In / Sign Up / Forgot Password — auth shell, midnight navy panel + warm form side
2. Dashboard Home — KPI grid + occupancy donut + revenue bar + events list + node status
3. Spaces Management — space grid, add/edit/delete, IoT node association flow
4. Reservations — table with status chips, details dialog
5. Events / IoT Alerts — filterable event log with severity chips
6. Camera Monitor — live feed placeholder
7. Operations History — audit table
8. Profile / Settings — owner profile form
